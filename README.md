# AstrBot 群成员信息导出插件

## 功能简介

本插件用于导出QQ群成员信息到Excel表格，包含成员昵称、QQ号、加入时间等详细信息。

## 使用说明

在QQ群聊中发送命令：
```
/导出群数据
```

插件将自动生成Excel文件并发送到群聊中。

## 注意事项

- 仅支持QQ平台
- 生成的Excel文件会保存在插件`temp`目录下
